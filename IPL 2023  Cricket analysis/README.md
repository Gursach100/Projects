# IPL 2023 Cricket Analysis

This project analyses IPL 2023 match and player performance data using Power BI to uncover insights into team performance, player statistics, and match outcomes.

## Project Objective

The goal of this project was to explore IPL 2023 data and build an interactive dashboard that highlights key cricket analytics including:

- Player performance
- Team statistics
- Match results
- Batting and bowling insights

## Tools Used

- Power BI
- Excel
- Data Cleaning
- Data Visualization

## Project Structure

Data/  
Contains the raw datasets used for the analysis.

Power BI File/  
Includes the Power BI dashboard (.pbix file).

Background Images/  
Images used for dashboard design and visuals.

Other Files/  
Supporting resources used during the project.

Report/  
Project report summarizing findings and insights.

## Key Insights

- Identified the **Top 5 batters of IPL 2023** based on total runs scored, batting strike rate, and overall consistency across matches.

- Analysed bowling performance to determine the **Top 5 bowlers of the tournament** using metrics such as total wickets taken, bowling economy, and match impact.

- Built a **Team of the Tournament** by selecting the best-performing players across batting, bowling, and all-round performance.

- Developed interactive Power BI dashboards to visualise player performance, team comparisons, and tournament statistics.
